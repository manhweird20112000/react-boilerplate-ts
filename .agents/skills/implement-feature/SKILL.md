## Steps to Implement a New Feature

Thực hiện các bước sau để triển khai một tính năng trong codebase:

1. Thực hiện define các interface cần thiết cho tính năng mới.
2. Tạo các api service để giao tiếp với backend.
3. Tạo các hook tùy chỉnh để quản lý state và logic liên quan đến tính năng.
4. Tạo các UI features để hiển thị và tương tác với người dùng.
5. Ghép logic và UI lại với nhau để hoàn thiện tính năng.
6. Viết unit tests và integration tests để đảm bảo tính năng hoạt động đúng.
7. Kiểm tra và debug tính năng để đảm bảo không có lỗi.

Không được bỏ qua một bước nào trong quá trình triển khai để đảm bảo tính năng được hoàn thiện và hoạt động tốt.
